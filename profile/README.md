## What is Neomailbox?

Neomailbox is a privacy-enhanced, secure email service that has been operating for over 23 years [citation:9][citation:12]. Built and operated by security professionals, the service is hosted entirely in Switzerland, offering strong privacy protection under one of the world's most independent jurisdictions [citation:2][citation:12]. Unlike mainstream providers, Neomailbox includes unlimited aliases with every account, supports OpenPGP encryption, and offers anonymous payment options including cryptocurrency and cash by mail [citation:2]. The service automatically strips IP address data from emails, protecting your location and identity [citation:12].

Infrastructure teams managing secure communications benefit from Neomailbox's multi-level security design and offshore hosting [citation:12]. System administrators appreciate full SSL-secured IMAP, POP3, and SMTP support, making integration with Outlook, Thunderbird, and any other email client on Windows straightforward.

## Setup & Configuration

### Setting Up on Windows

Neomailbox works with any standard email client on Windows, including Outlook, Thunderbird, and Windows Mail.

### Option 1: Generic IMAP/SMTP Configuration

**IMAP Settings (Incoming Mail):**

| Setting | Value |
|---------|-------|
| Server | `neomailbox.net` (use `.com` for US-hosted legacy accounts) [citation:6] |
| Port | `993` (SSL required) [citation:6] |
| Authentication | Password authentication |
| Username | Your Neomailbox username |
| Password | Your Neomailbox password |

**SMTP Settings (Outgoing Mail):**

| Setting | Value |
|---------|-------|
| Server | `smtp.neomailbox.net` (use `.com` for US-hosted legacy accounts) [citation:6] |
| Port | `587` (STARTTLS) or `465` / `125` (SSL) [citation:6] |
| Authentication | Password authentication |
| Username | Your Neomailbox username |
| Password | Your Neomailbox password |

### Option 2: Outlook Configuration

**Step 1:** Open Outlook and navigate to **File** > **Add Account**.

**Step 2:** Enter your Neomailbox email address and select **IMAP** as the account type.

**Step 3:** Enter your Neomailbox password.

**Step 4:** Configure incoming and outgoing mail settings using the generic settings above.

**Important:** Do not enable Secure Password Authentication (SPA) in Outlook [citation:6][citation:7].

### Option 3: Webmail Access

Neomailbox provides a dynamic, multi-lingual webmail system that works in any modern browser [citation:12].
